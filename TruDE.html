TruDE

Trude's Desktop Environment

What is TruDE?

TruDE is an open source desktop environment for Linux, designed to be extremely lightweight and customizable.

Automate your workflow

Window tiling is done automatically

Focus on what really matters and let TruDE do the rest. Window tiling is automatic and fully customizable.

Fast and lightweight

The window manager is modular and extremely lightweight. TruDE is written in under 3000 lines of code, while supporting transparency, blur and any layout imaginable.

Workspaces

TruDE supports various workspaces: Virtual desktops that can hold any layout, apps and different behaviours. It's the perfect solution for multi-tasking and efficiency, while still keeping everything simple.

Keyboard navigation

Every action can be performed with the keyboard, using VIM-like shortcuts by default.

Works everywhere

Any machine capable of running C can use TruDE. Most modern CPU architectures are supported, including ARM, RISC-V and x86.

Open nature

Don't like it? Change it. The code is fully open-source, distributed under the GPL-3 license.
