A new IDE - NeoPrism

NeoPrism was created to provide a lightweight development experience with integration with other tools and additional features.

Components and functions

Plugin Manager

Este componente carrega, verifica e instala todos os outros componentes e plugins.

Dashboard

O NeoPrism possui uma tela de inicio que tráz diversas vantagens de utilização.

Fuzzy Finder

O NeoPrism possui um componente que fornece recursos avançados de pesquisa, navegação e seleção de dados numa pasta ou ficheiro.

LSP / CMP

A IDE fornece recursos avançados de edição e autocompletação de código em projetos.

Treesitter

Este também oferece recursos avançados de análise de sintaxe e estruturação de código. Ele traz ao utilizador recursos como a capacidade de identificar com precisão as partes do código-fonte e a sua estrutura.

Bufferline

Este componente oferece recursos avançados de gestão e visualização de buffers.

File Explorer

A Ide contém um explorador de ficheiros que permite a visualização da estrutura de diretórios do sistema de arquivos do computador diretamente no editor de texto.

Terminal

O NeoPrism permite que o utilizador crie terminais exibido de diferentes formas, dependendo da escolha do mesmo no próprio editor.
